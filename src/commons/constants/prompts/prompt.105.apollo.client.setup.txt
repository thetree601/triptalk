핵심요구사항) Apollo Client를 설정하고 Provider를 구현할 것.
            1. Apollo Client 설정:
                - GraphQL 엔드포인트: https://main-practice.codebootcamp.co.kr/graphql
                - InMemoryCache 사용
                - HTTP Link 설정
                - 에러 처리 포함
                
            2. Apollo Provider 구현:
                - "use client" 지시어 사용
                - ReactNode children props 받기
                - ApolloProvider로 감싸기
                - TypeScript 타입 정의 포함
                
            3. Layout 통합 준비:
                - 기존 ReactQueryProvider와 함께 사용할 수 있도록 설계
                - Provider 순서: ApolloProvider > ReactQueryProvider > ModalProvider
                
            4. 캐시 설정:
                - Board 타입 캐시 정책 설정
                - 페이지네이션 캐시 정책
                - 인기 게시글 캐시 정책
                
            5. 에러 처리:
                - GraphQL 에러 처리
                - 네트워크 에러 처리
                - 사용자 친화적 에러 메시지
                
            6. 실제 스키마 연동:
                - fetchBoards 쿼리 지원
                - fetchBoardsOfTheBest 쿼리 지원
                - Board 타입 정의 포함
